  На клиенте

    выполнение index.html
    запрос-получение дополнительных файлов (скрипты, стили)
    выполнение этих файлов

    запрос на сервер "давай данные"-
      и обработчик для выполнения ответа, когда он придёт (асинхронно)
      -в конце разрешить refresh.

      пользователь в это время уже получает возможность действий на странице
      (которые нужно контролировать, чтобы он не сломал всё)

    кнопки на странице:

      rfresh - disable, enable автоматом после завершения  обработчика

      config (меню вида таблицы) - enable
        (если пользователь что-то поменяет в виде таблицы, то
          пере-послать запрос на сервер c учётом изменённого (строка 12)

      add - (((нельзя!-enable (если пользователь что-то добавит в таблицу, то
          пере-послать запрос на сервер c учётом добавленного (строка 12))
          НАДО: disable, enable автоматом после завершения обработчика
